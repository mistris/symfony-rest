# symfony-rest

#### User shopping experience features:
* Add or remove product
* Add or remove cart
* Add product to cart
* Remove product from cart
* Get all cart products
* Get cart subtotal
* Get cart VAT amount
* Get cart total

#### How to start
`./update.sh` - it will update permissions, update DB, clear cache

#### REST API documentation and sandbox
`/api/doc`

#### How to run tests
`./test.sh`
